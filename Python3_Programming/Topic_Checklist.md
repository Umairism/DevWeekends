# Python Mastery Checklist (Codebasics)

## 1. Environment Setup
- [ ] Install Python 3 and verify with `python --version`
- [ ] Run "Hello World" from terminal
- [ ] Run "Hello World" in PyCharm

## 2. Variables & Data Types
- [ ] Declare `int`, `float`, `str`, `bool`
- [ ] Print values with `type()`
- [ ] Understand basic type system

## 3. Operators
- [ ] Use `+`, `-`, `*`, `/`, `//`, `%`, `**`
- [ ] Explain difference between `/` and `//`

## 4. Strings
- [ ] Index characters in a string
- [ ] Slice substrings
- [ ] Use string methods: `upper()`, `lower()`, `strip()`, `replace()`, `split()`
- [ ] Format strings using f-strings and `.format()`
- [ ] **Mini Project:** Capitalize each word in a sentence

## 5. Lists
- [ ] Create list
- [ ] Use `append()`, `insert()`, `remove()`, `pop()`
- [ ] Iterate using `for` loop
- [ ] Access via index and negative index
- [ ] Sort, reverse, length
- [ ] Slicing lists
- [ ] **Mini Project:** Filter even numbers

## 6. Control Flow
- [ ] `if` / `elif` / `else` (number categorization)
- [ ] `for` loop with `range()` and lists
- [ ] `while` loop with `break` and `continue`
- [ ] **Mini Project:** FizzBuzz

## 7. Functions
- [ ] Define functions with parameters and return values
- [ ] Default arguments
- [ ] Keyword arguments
- [ ] Understand variable scope
- [ ] **Mini Project:** Factorial (loop-based)

## 8. Dictionaries
- [ ] Create dictionary
- [ ] Access, add, update, delete keys
- [ ] Iterate with `.keys()`, `.values()`, `.items()`
- [ ] **Mini Project:** Phone book system

## 9. Modules & Packages
- [ ] Use `math` module (`sqrt`, `ceil`, `floor`)
- [ ] Install external package using `pip`
- [ ] Create custom module and import it

## 10. File Handling
- [ ] Read file line by line
- [ ] Write list to file
- [ ] Read file back

## 11. JSON Handling
- [ ] Parse JSON using `json.loads()`
- [ ] Convert dict to JSON
- [ ] **Mini Project:** Student records (highest score)

## 12. Error Handling
- [ ] Handle `ZeroDivisionError` using `try`/`except`
- [ ] Use multiple `except` blocks
- [ ] Use `finally` block
- [ ] Raise custom exceptions

## 13. OOP (Object-Oriented Programming)
- [ ] Create class with `__init__`
- [ ] Add attributes and methods
- [ ] Create multiple instances
- [ ] Implement `__str__`
- [ ] **Mini Project:** BankAccount class

## 14. Inheritance
- [ ] Create child class
- [ ] Use `super()` in constructor
- [ ] Method overriding
- [ ] Multiple inheritance
- [ ] Understand MRO

## 15. Iterators & Generators
- [ ] Custom iterator (`__iter__`, `__next__`)
- [ ] Generator with `yield`

## 16. Advanced Python
- [ ] List comprehensions (filter, map, nested)
- [ ] Understand `if __name__ == "__main__":`

## 17. Final Project
- [ ] CLI Student Grade Manager
  - [ ] Add students
  - [ ] Record scores
  - [ ] Calculate average
