# My Python Learning Journey

Hey there! This is a log of my weekend project where I worked through the core concepts of Python. I wanted to keep track of what I learned and what gave me a bit of trouble along the way.

## Topics I Covered
I wrote separate code files to practice each of these hands-on:
- Environment Setup & Basics
- Variables, Data Types & Operators
- Strings & Lists (lots of slicing and indexing)
- Control Flow (`if/else`, `for` and `while` loops)
- Functions & Variable Scope
- Dictionaries (key-value pairs)
- Modules & Packages
- File Handling & JSON Parsing
- Error Handling (`try/except/finally`)
- Object-Oriented Programming (Classes, `__init__`, and Inheritance)
- Iterators & Generators (`yield`)
- List Comprehensions 
- **Final Project:** Built a CLI-based Student Grade Manager

## Difficulties I Faced
It wasn't a total breeze. Here are a few concepts that required some extra brain power:
- **Object-Oriented Programming:** Wrapping my head around `self`, the `__init__` constructor, and how inheritance passes methods down to child classes took some time to click.
- **Method Resolution Order (MRO):** Multiple inheritance is cool, but figuring out exactly which parent class method gets executed first was a bit tricky.
- **Generators vs Iterators:** Understanding how `yield` pauses a function's state (instead of just returning and destroying the local variables) was definitely a mind-bender.
- **JSON Parsing:** Kept getting tripped up remembering the difference between `json.loads()` (string to dict) and `json.dumps()` (dict to string). 

Overall, building the mini-projects and keeping the code clean really helped lock it all in!
